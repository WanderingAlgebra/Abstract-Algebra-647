# 抽象代数期末复习（647 版）

这份讲义笔记是笔者在复习抽象代数课程时，根据 B 站 up 主“我真的不懂分析”的网课自己整理的笔记，并由 AI 工具润色而来。

原网课顺序为：

1. 域和线性空间
2. 环和模
3. 群和群作用
4. Galois 理论

由于笔者所在学校不讲授也不考查 Galois 理论，这份笔记只计划记录前三章内容。本讲义更适合作为复习材料，或供此前对抽象代数已有一定了解的读者查漏补缺；不适合作为完全初学者的第一份入门材料。

讲义封面的插画来自Pixiv的画师Hiten。

## 编译方法

使用 XeLaTeX 编译，以保证中文字体和交叉引用正常：

```bash
xelatex -interaction=nonstopmode main.tex
xelatex -interaction=nonstopmode main.tex
```

第二遍编译用于刷新目录、定理编号和超链接。当前日志中可能保留宋体斜体替代警告，这是字体形状问题，不影响正文内容和链接。

## 当前内容

目前已整理第一章“域”，内容围绕域的基本概念、域同态、域的特征和域扩张展开。

第一章会从熟悉的数系出发，逐步讨论如何构造新域、两个域之间的同态有什么限制、特征如何区分不同类型的域，以及有限扩张、代数扩张、极小多项式和 Galois 群的入门概念。

代表例子包括 `\mathbb{Q}(\sqrt[3]{2})`、`\mathbb{Q}(\sqrt2,\sqrt3)`、无限生成扩张和有理函数域 `\mathbb{Q}(x)`。各节末附有少量课堂问题参考解答。
